# 人生追踪器 - Life Tracker

一个集成了人生倒计时、愿望清单和结绳记事功能的个人生活管理工具。

## 功能特性

### 1. 人生倒计时 ⏰
- 显示当前生命的剩余总时间（年/月/周/天）
- 显示距离下一年、下一月、下一周、下一天的倒计时
- 可自定义出生日期和预期寿命

### 2. 愿望清单 📝
- 创建多个愿望清单
- 每个清单可以添加多个愿望项目
- 支持标记完成状态（打勾）
- 每个项目可以记录文字笔记
- 支持编辑和删除清单/项目

### 3. 结绳记事 📅
- 记录每天发生的事情
- 支持查看历史记录
- 快速选择有记录的日期
- 所有记录保存在本地

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

然后在浏览器中打开 [http://localhost:3000](http://localhost:3000)

### 构建生产版本

```bash
npm run build
npm start
```

## 使用说明

### 人生倒计时
1. 在"出生日期"输入框中选择你的出生日期
2. 在"预期寿命"输入框中设置预期寿命（默认 80 年）
3. 系统会自动计算并显示：
   - 剩余总时间（年/月/周/天）
   - 距离下一年/月/周/天的倒计时

### 愿望清单
1. 点击"创建清单"按钮，输入清单名称
2. 在清单中添加愿望项目
3. 点击复选框标记完成状态
4. 点击"记录"按钮为项目添加文字笔记
5. 可以创建多个清单来分类管理不同的愿望

### 结绳记事
1. 切换到"今日记录"模式
2. 在文本框中记录今天发生的事情
3. 点击"保存今日记录"保存
4. 切换到"查看历史"模式可以查看和浏览历史记录
5. 点击日期按钮快速跳转到有记录的日期

## 数据存储

所有数据都保存在浏览器的本地存储（localStorage）中，不会上传到任何服务器，完全私密安全。

## 部署到 GitHub Pages

### 步骤

1. **将代码推送到 GitHub**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/你的用户名/你的仓库名.git
   git push -u origin main
   ```

2. **启用 GitHub Pages**
   - 在 GitHub 仓库中，进入 Settings → Pages
   - Source 选择 "GitHub Actions"
   - 保存设置

3. **自动部署**
   - 每次 push 代码到 main 分支，GitHub Actions 会自动构建并部署
   - 部署完成后，访问地址为：`https://你的用户名.github.io/你的仓库名/`

### 数据管理

- **数据存储**：所有数据保存在浏览器的 localStorage 中
- **数据导出**：在"数据管理"模块中点击"导出数据"，会下载 JSON 文件
- **数据导入**：可以导入之前导出的 JSON 文件恢复数据
- **备份建议**：定期导出数据并保存到 GitHub 仓库的 `backups/` 目录

### 注意事项

- 清除浏览器数据会丢失所有本地数据
- 不同浏览器/设备之间的数据不会自动同步
- 建议定期导出备份
